基于物理的渲染着色模型

1. 概述

在 2013 年的 SIGGRAPH 会议上，来自 Epic 的 Brian Karis 介绍了 Unreal Engine 4 中使用的一种基于物理的着色模型，这一着色模型可以在实时渲染的计算需求下达成良好的视觉效果，实现计算复杂度与显示效果的平衡，同时着色模型中的 metallic-roughness 工作流对对艺术家创作非常友好，使得整体开发流程灵活且高效，因而在此后多年逐渐成为业界广泛使用并不断扩展延伸的一个着色模型。

2. 渲染方程

2.1 定义

为了理解这一着色模型是如何实现良好的渲染效果的，我们需要从渲染方程来入手，其一般形式可以写作下式

(1)Lo(p,ωo)=Le(p,ωo)+∫Ωfr(p,ωi,ωo)Li(p,ωi)(n⋅ωi)dωi

  • Lo：点 p 在方向 ωo 的出射光

  • Le：点 p 的自发光（如果它是光源）

  • Li：从方向 ωi 入射的光

  • fr：双向反射函数BRDF，描述反射光如何形成

  • n⋅ωi：入射角余弦项，表示光线与表面法线的夹角影响

在实际计算过程中，对于这样的一个无法求解析解的定积分，我们往往采用朴实蒙特卡洛积分的思想，也就是可以将其简化为以下的求和形式

(2)Lo(p,ωo)≈Le(p,ωo)+1N∑i=1Nfr(p,ωi,ωo)Li(p,ωi)(n⋅ωi)

为了解得最终的 Lo ，我们需要对上面的求和形式的渲染方程进行拆解，拆解需要依据入射光 Li 的不同与双向反射函数 BRDF fr 的不同来进行。

2.2 入射光 Li

在忽略物体自发光的情况下，我们可以将着色点的出射辐照度来源区分为两种，即直接光照与间接光照

(3)Li=Ldirect+Lindirect

  • Ldirect ：直接光，即有明确数值与几何定义的光源，比如显式定义的方向光、点光等。在计算过程中，我们可以很轻松的获取着色点在在某一光源贡献下的入射辐照度与入射角度，从而带入计算。

  • Lindirect ：间接光，即没有明确定义的光源，可以认为是场景中整体的环境光照，在此着色模型中使用 IBL（Image Based Light） 方法来就行进行计算，用一张环境贴图来模拟整体的环境光，在流程中通过提前的预处理与运行时的查找表采样来获得对应的计算结果。

2.3 双向反射函数 BRDF fr

2.3.1 物理现象

当光击中物体时，会出现两种情况：一种是光被物体表面直接反射出去，没有进入物体内部；一种是光进入了物体内部后，与物体内部发生作用，其中又包含了两种情况，一部分光会被物体内部的自由电子吸收或者是原子结构吸收特定波长（产生颜色），其余部分会再次发生散射，称为次表面散射。对于直接发生的散射，我们可以使用 Cook-Torrance 镜面反射模型来进行计算，对于不透明物体上的次表面散射，可以用 Lambert 漫反射模型来进行近似计算。

对于半透明材质（皮肤、蜡），Lambert 近似会失真，必须用 SSS 专门模型。

2.3.2 Cook-Torrance BRDF

2.3.2.1 基本思想

Cook-Torrance 模型是一种基于微表面理论的与统计几何光学的模型，通过概率分布描述微观面元的朝向，1977年Blinn和1981年 Cook 和 Torrance 将微表面理论引入了计算机图形学中。这一着色模型的基本思想是：所有表面在微观尺度上都是凹凸不平的。因此，物体表面的粗糙程度将直接影响物体如何反射光线。基于此思想，Cook-Torrance 着色模型需要知道如何描述物体表面有多粗糙，从而解决粗糙程度如何影响光线的反射的核心问题。

2.3.2.2 着色计算

Cook-Torrance 着色模型的 BRDF 函数如下

(4)fcook-torrance=F(v,h)D(h)G(l,v)4(n⋅l)(n⋅v)

  • F 菲涅耳项，描述不同材质在不同角度下会有多少光线参与表面散射计算

  • D 法线分布函数，描述不同粗糙程度的物体表面有多少微观面元的朝向与观察方向一致

  • G 几何项，描述物体表面各个微观面元相互之间如何形成阴影与遮挡

  • v h l n 视角方向 半程向量 光源方向 表面法线

  • 4(n⋅l)(n⋅v) 归一化系数

2.3.3 Lambert BRDF

2.3.3.1 基本思想

Lambert 着色模型是一个漫反射光照的经验模型，他由约翰·海因里希·兰伯特（Johann Heinrich Lambert）提出，其基本思想是光线打在物体表面后会均匀的向各个方向反射出去，因此得到了以下的结论

  • 着色与观察角度无关（表面具有各向同性）

  • 着色与入射光线与法线的夹角有关

2.3.3.2 着色计算

基于 Lambert 定理的基本思想，可以通过计算证明得到 Lambert BRDF 为一个常数，与入射角、法线、观察方向等都没有关系

(5)flambert=ρπ

其中 ρ 满足 ρ∈[0,1] ，定义了有多少入射能量最终参与到反射中， π 是归一化系数。

3. 直接光照

对于具有显示定义的光源，我们可以根据不同的光源类型、用对应的方法来计算得到他们的入射辐照度 radiance = light.color 与几何信息光线入射角度 L = normalize(light.direction) ，进而带入渲染方程中进行计算得到入射光对在这一点上的着色结果，代码参考如下

// directional light
vec3 L = normalize(d_light.direction);
vec3 H = normalize(V + L);
vec3 radiance = d_light.color;
float bias = max(0.05 * (1.0 - dot(N, L)), 0.005);
float visibility = directional_shadow(world_space_position, bias);
Lo += direct_irradiance(radiance, albedo, V, N, L, F0, roughness, metallic) * visibility;
// point light
L = normalize(p_light.position - world_space_position);
H = normalize(V + L);
float d = distance(p_light.position, world_space_position);
float attenuation = 1.0 / (p_light.constant + d * p_light.linear + d * d * p_light.quadratic);
radiance = p_light.color * attenuation;
visibility = point_shadow(world_space_position, 0.015);// 点阴影用固定bias减少噪点
Lo += direct_irradiance(radiance, albedo, V, N, L, F0, roughness, metallic) * visibility;

代码中的 visibility 描述了光源对着色点的由于遮挡产生的可见性变化，用于实现阴影效果，一般可以使用 Shadow Map 算法来进行计算。

3.1 镜面反射

镜面反射的计算就是 Cook-Torrance 着色模型的 BRDF 的计算，根据上文可知，就是利用着色点的材质与几何相关的数据计算得到对应的 F ， D ，G 项，然后带入即可到对应的 BRDF 结果，参考代码如下

xxxxxxxxxx
// specular
float NDF = DistributionGGX(NdotH, roughness);
float G = GeometrySmith(NdotV, NdotL, roughness);
vec3 F = fresnelSchlick(HdotV, F0);
vec3 numerator = NDF * G * F;
float denominator = 4.0 * NdotV * NdotL + 0.0001;
vec3 specular = numerator / denominator;

3.1.1 菲涅尔项

菲涅尔项描述了有多少光在表面直接反射出去，也就是镜面反射系数 ks=F 。物理上用菲涅尔方程来对这一现象进行定量描述，但是完整的菲涅尔方程描述了光在不同折射率的介质界面上，反射光振幅与入射光振幅的关系。它非常复杂，不仅涉及折射率 n，还涉及光的偏振态（S偏振和P偏振），完整的数学表达非常复杂，非常不适合实时渲染中的计算，因此我们可以采用 Schlick 近似来描述镜面反射系数。 Schlick 近似公式计算如下

(6)F=F0+(1−F0)(1−cosθ)5

  • F0 ：基础反射率

此外，对于不同的材质，光线达到物体表面时有多少光背反射也是其固有的属性，也就是每个材质本身的基础反射率。对于非金属来说，这一系数是当光线从空气（折射率约为 1.0）垂直入射进入物体（折射率记为 n）时，由折射率公式 F0=(n−1n+1)2 计算得到。对于金属来说，由于涉及吸收系数 k，折射率计算极其复杂。因此工程中采取近似方案，直接让基础色（基础反射率albedo）来充当 F0。此外，对于一般物体，工程中还需要根据金属度来在非金属与金属之间进行线性插值。

xxxxxxxxxx
vec3 F0 = vec3(0.04);\\因为大多数非金属的反射差异极小 因此可以硬编码为0.04 是一个非常稳妥的中间值。
F0 = mix(F0, albedo, metallic);

  • θ ：入射角

请回忆起生活中的这样一个场景，当站在湖边俯视脚下的水面，我们可以看清透明的湖底，但如果我们望向远处的对岸，就只能看到远山的倒影，此时水面几乎是完全不透明的镜子。这正是菲涅尔现象所描述的：光与物体表面所成的夹角越小，也就是入射角度越倾斜，反射率就越高。当光垂直于入射表面时，几乎不发生反射，而当光以掠射角（几乎与表面平行，与法线夹角接近 90°）入射时，反射率几乎达到 100% ，因此，入射夹角对镜面反射系数有决定性作用。

Schlick 近似公式的代码参考如下

xxxxxxxxxx
vec3 fresnelSchlick(float cosTheta, vec3 F0)
{
    return F0 + (1.0 - F0) * pow(clamp(1.0 - cosTheta, 0.0, 1.0), 5.0);
}

3.1.2 法线分布函数

微表面模型认为，物体表面上只有一部分的微观表面反射出来的光会对最终的反射有贡献，当给定入射光和视线方向时，这部分有贡献的微表面的法线方向应与半程向量方向重合。通过法线分布函数 NDF ，可以对这些微表面法线方向的统计分布进行描述。

(7)D(H)=α2π[(N⋅H)2(α2−1)+1]2

  • α ：表面粗糙度 roughness 的平方，代表表面微面元的斜率分布

  • N⋅H 平面法线与半程向量夹角

代码参考如下

xxxxxxxxxx
float DistributionGGX(float NdotH, float roughness)
{
    float a = roughness*roughness;
    float a2 = a*a;
    float NdotH2 = NdotH*NdotH;
    float nom   = a2;
    float denom = (NdotH2 * (a2 - 1.0) + 1.0);
    denom = PI * denom * denom;
    return nom / denom;
}

3.1.3 几何遮蔽项

由于物体表面在微观尺度上的凹凸不平，光线在其反射时还必须考虑到表面的遮挡（Shadowing）与自遮挡（Masking）的现象，几何遮蔽项 G 正是用来描述这一现象的，它是一个 0 到 1 之间的值，1代表完全没有遮挡，光线全部通过；0则代表遮挡极其严重，没有光线能反射出来。

Shadowing是指入射光由于微表面的阻挡无法到达入射点的现象，Masking是指反射光由于微表面的阻挡无法出射的现象。形象的说，我们可以把微观表面想象为无数细小的“山峰”和“山谷”。当光线射入或我们进行观察时，Shadowing就是入射光线被微观表面的“山峰”挡住，无法照射到“山谷”中，导致能量无法到达着色点。Masking就是反射光线在飞向观察者的途中，被其他的微观面元挡住，导致观察者看不到这部分光。

UE4 的着色模型选择用 Smith-Schlick 模型来描述他，公式如下

(8)G=cos⁡θlcos⁡θv(cos⁡θl(1−k)+k)(cos⁡θv(1−k)+k)

这个公式其实是两个相同形式的函数的乘积，分别处理了光照方向与视角方向的遮挡

(9)G(l,v)=Gsub(l)⋅Gsub(v)

其中单体函数 Gsub 为

(10)Gsub(x)=cos⁡θxcos⁡θx(1−k)+k

  • k ：基于粗糙度 α 的重映射系数。它决定了表面有“多大可能性”产生遮挡，对直接光照来说，k=(α+1)28 。

  • θx ：光照/视角方向与法线的夹角，夹角越大，分母中的 k 占比就越大，导致 G 值迅速减小，模拟了边缘处更严重的遮挡。

在最终的实际直接光照计算过程中的，公式如下所示

(11)G(N,V,L)=GSchlickGGX(N⋅V,k)⋅GSchlickGGX(N⋅L,k)=N⋅V(N⋅V)(1−k)+k⋅N⋅L(N⋅L)(1−k)+k

代码参考如下

xxxxxxxxxx
float GeometrySchlickGGX(float NdotV, float roughness)
{
    float r = (roughness + 1.0);
    float k = (r*r) / 8.0;
    float nom   = NdotV;
    float denom = NdotV * (1.0 - k) + k;
    return nom / denom;
}
float GeometrySmith(float NdotV, float NdotL, float roughness)
{
    float ggx2 = GeometrySchlickGGX(NdotV, roughness);
    float ggx1 = GeometrySchlickGGX(NdotL, roughness);
    return ggx1 * ggx2;
}

3.2 漫反射

现在我们再来考虑一下漫反射的光。对于金属来说，光进入物体内部后，由于其内部充满了高密度的自由电子，光波进入后会引起自由电子振动，光能几乎瞬间被吸收，转化为热能，此时漫反射系数 kd=0 ；对于非金属来说， 内部没有自由电子，此时可以根据能量守恒计算得到漫反射系数 kd=1−F 。最终结合金属与非金属的现象，我们可以得到一般物体的漫反射系数如下

(12)kd=(1−F)(1.0−metallic)

光进入物体内部后，除了被自由电子吸收以外，还会在分子间反复碰撞、散射，在这个过程中根据材质的不同，某些波长的光被吸收，从而决定了物体的颜色，因此物体材质同决定了有多少能量会被漫反射出去，最终漫反射率的常数计算如下

(13)ρ=kd⋅albedo

结合归一化常数 π ，可以得到计算漫反射 BRDF 代码参考如下

xxxxxxxxxx
// diffuse
vec3 kS = F;
vec3 kD = (vec3(1.0) - kS) * (1.0 - metalness);
vec3 diffuse = kD * albedo / PI;

3.3 最终结果

对于给定的入射光，我们已经分别计算得到了其在指定着色点上的镜面反射与漫反射的 BRDF ，带入到渲染方程中即可计算出这个入射光对最终出射光的贡献，也就是 Lo=(fCook−Torrance+fLambert)⋅Li⋅cosθ ，最终直接光照的结果计算代码参考如下

{
