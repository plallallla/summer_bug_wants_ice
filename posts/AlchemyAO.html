AlchemyAO

1. 概述

AO（Ambient Occlusion，环境光遮蔽）是对场景中物体表面非直接光照的一个修正，他的基本思想是为每一个着色点估算一个被遮挡系数，他是一个 [0,1] 的值，以此来决定当前着色点能被环境光照亮到什么程度。AO的计算有多种方式，本文将介绍Alchemy风格的AO计算，这是一个由 NVIDIA提出的融合了 Morgan McGuire 的 Scalable Ambient Occlusion (SAO) 思想与一种新颖的遮蔽估算公式，在保持实时性能的同时显著提升了视觉质量、稳定性和物理合理性。AO计算的基本流程参考如下，接下来将介绍AlchemyAO的创新与优化。

2. 采样

要实现良好的AO计算效果，我们希望可以高效、低噪声地分布采样点，采样点生成代码参考如下，其中传入参数为采样半径与采样点序列号

xxxxxxxxxx
vec3 get_sample_pt(float radius, int i)
{
    float alpha = 1.0 / sample_ct * (i + 0.5);
    float h = radius * alpha;
    float theta = TAU * alpha * sample_turns;
    vec2 u = vec2(cos(theta), sin(theta));
    int m = clamp(findMSB(int(h)) - 4, 0, max_mip); // McGuire paper MIP calculation
    ivec2 mip_pos = clamp((ivec2(h * u) + ivec2(gl_FragCoord.xy)) >> m, ivec2(0), textureSize(s_position, m) - ivec2(1));
    vec3 worldPi = texelFetch(s_position, mip_pos, m).xyz;
    vec3 Pi = vec3(position_transform * vec4(worldPi, 1.0));
    return Pi;
}

2.1 阿基米德螺线采样

代码中通过阿基米德螺线公式 r=a+bθ 计算了这样的两个变量

(1)ri=radius⋅αi,θi=2π⋅sample_turns⋅αi

其中 αi 随着采样点的增加而线性增加，也就可以获得同样线性增加的采样半径 ri 与采样偏转角 θi ，由此实现在圆形/圆盘区域内一组 2D 采样方向的生成。对比其他随机生成采样的方法，阿基米德螺线采样主要有以下优点

2.2 LOD计算与Mipmap采样

接下来在屏幕空间 AO 计算中，我们需要根据根据采样点半径 h（该值源于view/world space距离，并经透视校正），自动选择合适分辨率的位置缓冲（position buffer），以实现“尺度自适应”的采样，从而显著抑制噪声、走样和闪烁。也就是通过近处采样高分辨率、远处采样低分辨率低方式来获取平滑的结果。代码中的findMSB(h)是在计算h的最高非0位，相当于 ⌊log2⁡(h)⌋ ，从而确定Mipmap层级（对应mipmap每层分辨率减半）。这一方法被Morgan McGuire在Scalable Ambient Obscurance一文中提出。

3. 遮蔽估算公式

获得采样点后，可以用计算每个采样方向对总遮蔽量（occlusion）的贡献，计算公式如下

(2)contribution=max(0, V⋅N+bias)|V|2+ε

  • V⋅N=|V|cos⁡θ，其中 θ 是 V 与法线 N 的夹角

  • max(0, ...)，自动忽略背面的采样点

  • bias ，偏移值、用来防止产生自遮挡（Self-Occlusion）

  • |V|2+ε，通过引入距离平凡的衰减，来模拟距离越近的物体，对 AO 贡献越大；远处物体影响微弱，ε确保不除0

这个公式不仅实现了半球自动约束，而且有着自然的距离衰减与抗自遮挡效果。他可以看作真实AO计算（半球上的可见性积分） AO(p)=1π∫Ω+V(p,ω)dω 的启发式近似，通过高效计算从而获得视觉上高度合理的结果。

最终公式计算的代码参考如下

vec3 sample_pt = get_sample_pt(radius, i);
vec3 V = sample_pt - P;
float VdotN = dot(V, N) + sao_bias;
float VdotV = dot(V, V);
// AlchemyAO obscurance estimator : max(0, dot(V, N) + bias) / (|V|² + ε)
sao_occlusion += max(0.0, VdotN) / (VdotV + sao_epsilon);
