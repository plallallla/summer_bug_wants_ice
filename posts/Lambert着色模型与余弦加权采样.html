                  • Lambert 着色模型与余弦加权采样

                  1. 基本思想

                  Lambert 着色模型是一个漫反射光照的经验模型，他由约翰·海因里希·兰伯特（Johann Heinrich Lambert）提出，其基本思想是光线打在物体表面后会均匀的向各个方向反射出去，因此得到了以下的结论

                  • 着色与观察角度无关（表面具有各向同性）

                  • 着色与入射光线与法线的夹角有关

                  2. 着色计算

                  由于反射过程中一定是能量守恒的，也就是出射 irradiance 一定不大于入射 irradiance ，我们可以定义一个反射率的常数 ρ∈[0,1] 使得下式成立

                  (1)M=ρ⋅E

                  • E 是入射 irradiance，满足 E=∫Ω+Li(ωi)(n⋅ωi)dωi

                  • M 是出射 irradiance，满足 M=∫Ω+Lo(ωo)(n⋅ωo)dωo

                  Lambert 着色模型的基本思想，可以推导出以下结论

                  • 与观察角度无关 ⇒ Lo(ωo)=Lo 为常量

                  • 与入射光线与法线的夹角有关 ⇒反射能量正比于 cos⁡θi

                  将常数 Lo 提取出来，可以整理如下

                  (2)M=∫Ω+Lo(ωo)cos⁡θodωo=Lo∫Ω+cos⁡θodωo

                  接下来考虑半球面上的余弦函数的积分，计算如下

                  (3)∫Ω+cos⁡θdω=∫02π∫0π/2cos⁡θsin⁡θdθdϕ=∫02πdϕ∫0π/2sin⁡θdsin⁡θ=2π⋅12=π

                  由于Lambert 着色模型表面的各项同性，那么BRDF fr(ωi,ωo) 应满足 fr(ωi,ωo)=fr 为常数，此时可以计算 Lo 如下

                  (4)Lo=Lo(ωo)=∫Ω+fr(ωi,ωo)Li(ωi)(n⋅ωi)dωi=fr∫Ω+Li(ωi)(n⋅ωi)dωi=fr⋅E

                  将以上的结论带回 M 的计算可得

                  (5)M=frπ⋅E

                  由此计算得到 Lambert 着色模型的 BRDF 函数为

                  (6)fr=ρπ

                  3. 余弦加权采样

                  3.1 计算概率密度函数

                  考虑蒙特卡洛积分中的被积函数如下

                  (7)Lo=∫Ω+fr(ωi,ωo)Li(ωi)(n⋅ωi)dωi

                  根据之前的推导我们已经明确 BRDF 函数为常数 ρπ ，同时入射 Li 一般也是未知的，因此我们应该构造的 PDF 函数应该匹配剩下的几何项：n⋅ωi=cosθ ，也就是说概率密度正比于入射方向与法线夹角的余弦

                  (8)p(ω)∝cos⁡θ,ω∈Ω+

                  接下来求解归一化常数 C ，计算如下

                  (9)∫Ω+p(ω)dω=1⇒∫Ω+Ccos⁡θdω=C∫Ω+cos⁡θdω=1∫Ω+cos⁡θdω=π⇒C⋅π=1⇒C=1π

                  最终我们得到的 PDF 函数为

                  (10)p(ω)=cos⁡θπ,ω∈Ω+

                  3.2 测度分离

                  由于上式测度为方向 ω ，他属于二维流形（单位半球），其自然参数化是两个角度 (θ,ϕ)。而逆变换采样仅适用于一维连续分布，因此我们首先需要将 PDF p(ω) 表达为关于 (θ,ϕ) 的联合密度。考虑立体角微元为 dω=sin⁡θdθdϕ ，那么 PDF函数一定满足以下的式子

                  (11)p(ω)dω=cos⁡θπdω=cos⁡θπsin⁡θdθdϕ

                  可以看到之前求得的密度函数 PDF 相对于测度 dθdϕ 有联合概率密度函数如下

                  (12)p(θ,ϕ)=cos⁡θsin⁡θπ,θ∈[0,π2], ϕ∈[0,2π)

                  由于 ϕ 与是 θ 独立的且 ϕ 应满足均匀分布（ϕ不对PDF造成影响），我们可以分离出这两个测度各自的概率密度

                  (13)p(θ,ϕ)=pθ(θ)⋅pϕ(ϕ)

                  其中 θ 的 PDF 如下

                  (14)pθ(θ)=2cos⁡θsin⁡θ,θ∈[0,π2]

                  ϕ 的 PDF 如下

                  (15)pϕ=12π,ϕ∈[0,2π)

                  3.3 逆变换采样

                  首先计算 θ 的 CDF

                  (16)Pθ(θ)=∫0θpθ(t)dt=∫0θ2cos⁡tsin⁡tdt=∫0θsin⁡(2t)dt=[−cos⁡(2t)2]0θ=sin2⁡θ,θ∈[0,π2]

                  接下来求反函数，设 uθ∼U(0,1) ，对应下式有

                  (17)uθ=Pθ(θ)=sin2⁡θ⇒sin⁡θ=uθ⇒θ=arcsin⁡(uθ)

                  由于 ϕ 均匀分布，因此对于 uϕ∼U(0,1) 来说有下式成立

                  (18)pϕ(ϕ)=12π⇒ϕ=2πuϕ

                  3.4 生成采样结果

                  上一步我们得到了得到将均匀分布 u 映射到目标分布的采样器 θ=arcsin⁡(uθ) 与 ϕ=2πuϕ 。但构建的目标 PDF 为 p(ω) ，因此我们需要用 θ 与 ϕ 来表示 ω ，由于对 ω 进行采样就是对单位半球上的点进行采样，因此我们可以考虑将单位半球上一点坐标 (x,y,z) 表示成如下的形式

                  (19)x=sin⁡θcos⁡ϕ=uθcos⁡(2πuϕ)y=sin⁡θsin⁡ϕ=uθsin⁡(2πuϕ)z=cos⁡θ=1−uθ

                  这里需要注意，此处的点 (x,y,z) 的参考坐标是理想空间下的坐标，这个空间坐标的 Z 轴方向为 (0,0,1) ，而要将采样器生成的结果带入到渲染方程中计算，则需要在法线 n 与 Z 轴重合的坐标空间下进行，因此我们对这个点进行变换，将其转换到世界坐标中。

                  我们的目标 PDF p(ω)=n⋅ωπ,ω∈Ω+(n) 中的 Ω+(n) 表示以法线 n 为极轴的上半球，而在采样结果生成时，为了计算方便我们将问题简化到标准局部坐标系（nlocal=(0,0,1)），即 z 轴向上。在这个局部系中，我们通过逆变换采样得到：

                  (20)ωlocal=[xyz]=[uθcos⁡(2πuϕ)uθsin⁡(2πuϕ)1−uθ]

                  这可以表示单位半球 Ω+ 上的一点，也就是一个单位向量，满足 ∥ωlocal∥=1，且 z≥0。此时 ωlocal 是相对于“法线朝 z 轴”的理想化表面的方向。而在真实场景中，表面法线 n 是一个世界坐标系下的单位向量，比如 n=(0.3,0.5,0.8)。渲染方程中的入射方向 ωi、出射方向 ωo、BRDF、光照等全部在世界坐标系中定义。因此，不能直接使用 ωlocal，因为它假设法线是 (0,0,1)。

                  首先，目标坐标空间下 Z 轴与法线 n 重合，我们可以直接取 n 为 Z 轴，接下来找到切线与副切线并绑定到 XY 轴即可。记 v 是一个不与法线 n 共线的向量，则向量 t=normalize(n×v) 必定与法线垂直，可以作为切线向量，副切线向量一样可以通过法线与切线叉得到 b=normalize(n×t) ，我们可以得到变换记作 M=[tbn] ，最终输出的采样结果为

                  (21)ωworld=M⋅ωlocal=xt+yb+zn

                  注意，这里的正交基（Orthogonal Normalized Basis）是不唯一的，因为应用蒙特卡洛估计的渲染时，只要分布一致，具体 ONB 方向不影响统计正确性，也就是说本质上ONB只是帮我们确定一个法线与他的切平面，因此可以绕法向量 n 任意旋转 XY 轴，旋转后得到的坐标空间仍然有效。

                  3.5 示例代码

                  xxxxxxxxxx
                  // 半球上的余弦加权采样
                  glm::vec3 cosine_weighted_random_hemisphere(const glm::vec3& normal)
                  {
                      // Step 1: 生成两个随机数作为采样器输入
                      float u_theta = get_float(.0f, 1.0f);
                      float u_phi = get_float(.0f, 1.0f);
                      // Step 2: 计算局部坐标系中的余弦加权采样
                      float r = std::sqrt(u_theta);
                      float phi = 2.0f * pi * u_phi;
                      float x = r * std::cos(phi);
                      float y = r * std::sin(phi);
                      float z = std::sqrt(std::max(0.0f, 1.0f - u_theta));
                      // Step 3: 构建ONB并计算最终采样结果
                      glm::vec3 n = glm::normalize(normal);
                      glm::vec3 v = std::abs(n.x) > 0.9f ? glm::vec3(0.0f, 1.0f, 0.0f) : glm::vec3(1.0f, 0.0f, 0.0f);
                      glm::vec3 t = glm::normalize(glm::cross(v, n));
                      glm::vec3 b = glm::cross(n, t);
                      return x * t + y * b + z * n;
                  }
